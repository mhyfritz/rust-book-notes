# cargo commands

Action command
Create project cargo new <project-name>

Build for release cargo build --release
Build and run cargo run
Check code cargo check

# Cargo.toml

Cargo.toml lists project's metadata, e.g. name and version and its dependencies.

[package]
name = "hello_cargo"
version = "0.1.0"
authors = ["Your Name <you@example.com>"]
edition = "2018"

[dependencies]

# Release build

  • cargo build --release will compile with optimizations which will lead to faster execution, but slower compilation
  • exectuable is created in target/release instead of target/debug
